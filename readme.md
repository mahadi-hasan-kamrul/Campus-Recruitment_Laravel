# Student Guide DIU
## Project Overview
Student Guide DIUI is a comprehensive web-based collaborative platform designed to enhance academic resource management and knowledge sharing for students and faculty members of Daffodil International University.

## Project Description
Student Guide DIU is an innovative academic archive system developed as a final year undergraduate project in the Department of Computer Science and Engineering. This platform addresses critical challenges in educational resource distribution by providing a centralized repository for course materials, previous examination papers, video resources, and collaborative tools.

The system facilitates seamless interaction between students and faculty members while maintaining organized, semester-wise academic content tailored to both day and evening programs.

## Installation
### Prerequisites
- Web server with PHP support (Apache recommended)
- MySQL database server
- Modern web browser

### Setup Instructions
1. Clone the repository

2. Configure the database
- Create a new MySQL database named st_guide
- Import the provided SQL schema file

3. Configure server settings
- Update database connection parameters
- Set appropriate file upload permissions

4.Access the application
- Navigate to the project directory via web browser
- Complete the registration process using DIU email credentials
